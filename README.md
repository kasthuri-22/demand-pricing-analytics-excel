# Demand & Pricing Analytics using Excel

This project analyzes the relationship between product price and customer demand using regression analysis.

## Tools Used
- Microsoft Excel
- Regression Analysis
- Data Visualization

## Key Analysis
- Demand prediction using linear regression
- Revenue optimization based on price levels
- Visualization of price vs demand trends

## Key Insights
- Demand decreases as price increases, confirming price sensitivity.
- The regression model explains 99.8% of demand variation.
- Optimal price range identified between 60–65 to maximize revenue.
